NashornMapBackedGraphNode

Related Docs: object NashornMapBackedGraphNode | package interop

class NashornMapBackedGraphNode extends GraphNode with JsonableProxy

Object used within the JVM whose implementation of the GraphNode interface is based on navigation of a graph defined in JavaScript object or data structure. Backed by a Map that can include simple properties or Nashorn ScriptObjectMirror in the event of nesting. Handles cycles if references are provided.
