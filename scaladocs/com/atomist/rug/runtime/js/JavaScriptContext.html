class JavaScriptContext extends LazyLogging

Context superclass for evaluating JavaScript. Creates a Nashorn ScriptEngineManager and can evaluate files and JavaScript fragments, exposing the known vars in a typesafe way so we partly avoid the horrific detyped Nashorn API.

One of these per rug please, or else they may stomp on one-another
